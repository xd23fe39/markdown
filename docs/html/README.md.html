Aufruf

Das MARKDOWN Helper Tool mdhelper ist ein bash-Script zur einfachen Verwaltung von Markdown Dokumenten und der Konvertierung via pandoc zu HTML.

MARKDOWN Helper (mdhelper) Version 1.0

  Usage: mdhelper init|status|build

  Document converter tool: /usr/bin/pandoc

Ein einfacher Workflow mit mdhelper

mkdir ./docs
cd ./docs
mdhelper init
mdhelper build
